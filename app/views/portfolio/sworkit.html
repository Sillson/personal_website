Sworkit Exercise¶

Goal: Predict what actions are lead indicators of specific actions or workouts that might show a pattern of retaining more users.

Variables:¶

  • Sign-up date
  • Age
  • Gender
  • Workouts completed (type, date, frequency)
  • Current activity level (absolute beginner, beginner, intermediate, advanced)
  • If they came in from any specific advertising campaign
  • Plan they selected or didn't select
  • General fitness goal
  • If they have created and completed any custom workouts

Mock data set¶

Above we can see a mock data set with some of the attributes that Sworkit collects on it's users.

Missing information¶

There are a few things I'd like to note:

Activity Level: How is this calculated? Is it tabulated off of workouts completed or self-reported? Polynomial classification?

Campaign: Ideally this would be a foreign key to a table that had a lot of contextual information about the campaign -- i.e. google, bing, fb, display/search, keywords/ads associated, etc.

Custom Workouts: Ideally this would be a foreign key to a table that had contextual information about the custom workouts. Simply classifying the level of engagement of having a custom workout could be a binomial feature on the model, however there is probably a lot of additional information about the characteristics of a custom workout that could be used to infer more about a user's engagement.

Plan: 'Plan they selected or didn't select'. I'm not sure if this relates to Sworkit's 'Leaner, Fitter, Stronger' plans per your site or tier of service. I'm going to assume this is tier of service, and I'll attribute this as a set of 3, which would be foreign keys relating to free-trial, paid-standard, or paid-premium.

Fitness Goal: This is what I'll tabulate as Sworkit's plan offerings, as Leaner, Fitter, Stronger, or Other.

Basic Correlation Example¶

The nice thing about Pandas is that is allows us to do some pretty basic pairwise correlation of columns right out of the box between two columns in this data set.

For example, let's pretend we want to examine the relationship between workouts completed and age. Is there a correlation between workouts completed and the age of a user? Below I'm going to run a Pearson correlation between the age and workouts completed columns. Since the data that was inserted was random, we should not see a correlation coefficient of any great signifigance.

For reference: 1 is a complete positive correlation and -1 is a complete negative correlation

In [27]:
df['age'].corr(df['workouts_completed'], method='pearson')

As expected, there was an incredibly small correlation coefficient in the random data.

In order to identify a pattern of retaining more users, we'll have to define some metris for what retainment entails. We'll need to understand length of account being open, and interaction with account at the least.

To define those, we'll add a deleted_account variable to our dataset, and add time range for how long the account has been open

In [58]:
df = df.drop(columns=['age_log', 'workouts_completed_log', 'age_group', 'workout_group'])
